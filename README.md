# 📖 About the Project

Customer churn is one of the biggest challenges faced by subscription-based businesses. Acquiring a new customer costs significantly more than retaining an existing one, making churn prediction an essential business strategy.

This project leverages **Machine Learning** to analyze customer demographics, subscription details, billing information, and service usage to predict whether a customer is likely to discontinue their service.

The project demonstrates the complete Data Science workflow, including **data preprocessing, exploratory data analysis (EDA), feature engineering, model building, and performance evaluation.**

# 📊 Dataset

**Dataset:** Telco Customer Churn Dataset

The dataset contains customer information including:

- 👤 Customer Demographics
- 📞 Phone Services
- 🌐 Internet Services
- 📅 Contract Details
- 💳 Payment Methods
- 💰 Monthly Charges
- 💵 Total Charges
- ⏳ Customer Tenure
- 🚪 Churn Status (Target Variable)
